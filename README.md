# QuietSpace Finder — Prototype

This repository will become the QuietSpace Finder prototype: a small web app to help people find quiet public spaces and check in to indicate crowd levels.

This is an initial minimal commit to get you started. Customize the files and we'll build features step-by-step.

How to run locally

1. Clone the repo (if you haven't already):
   git clone https://github.com/arissC6/QuiteSpace-Finder.git

2. Open index.html in your browser, or use a simple static server for better geolocation support:
   - Python 3: `python -m http.server 8000`
   - Node (http-server): `npx http-server -p 8000`

3. Start editing index.html and commit changes. I'll help with the next files when you're ready.
